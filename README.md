# ice-breaker
AI-powered networking tool to facilitate engaging discussions at in-person and remote events.

## Installation
```cd ./ice-breaker```

```pip install pipenv```

```pipenv install```

## start web server
```cd ./ice-breaker/web/icebreaker```

``` npm install ```

``` npm start ```
